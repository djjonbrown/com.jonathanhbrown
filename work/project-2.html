Product Management, Product Design, UX Design, IxD
Project Overview
Wave Sciences solved one of the most confounding problems in audio with a revolutionary machine learning engine that provides an order-of-magnitude better performance than the state-of-the-art. However, the interface, called Acoustical Focus,  requires the user to be an expert forensic audiologist. To capture a broader user base, the interface needed to focus on a different type of user.
My Contributions
I developed a user-focused product plan that established a quarterly cadence for delivering real user value across a portfolio of three products. Before each quarterly increment, I followed a rapid iteration process of Observe - Reflect - Make to converge on the cross-product solution. During the quarter, I designed and refined the user-facing solution, working one sprint ahead of the development team.
